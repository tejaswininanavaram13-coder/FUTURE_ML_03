# 📄 Resume Screening System  

An intelligent **Machine Learning-based Resume Screening System** that analyzes resumes against job descriptions and provides a match score, ranking, and missing skills.

---

## 🚀 Features

- 📊 Resume–Job Description Matching Score  
- 🧠 Skill Extraction from Resume & JD  
- ⚠ Missing Skills Detection  
- 🏆 Candidate Ranking (Highly / Moderate / Not Suitable)  
- 📄 Dual Input Support:
  - Paste Resume Text  
  - Upload Resume (PDF)  
- 🎨 Modern UI with clean design  

---

## 🛠 Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- NLP (Text Processing)  
- PyPDF2
